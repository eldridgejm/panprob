Exceptions

All exceptions derive from panprob.exceptions.Error:

class panprob.exceptions.Error

Base class for exceptions in this module.

More specific exceptions are:

class panprob.exceptions.ParseError

Raised when a parser encounters an error.

class panprob.exceptions.RenderError

Raised when a renderer encounters an error.

class panprob.exceptions.IllegalChild(parent, child)

Raised when attempting to add a child node of a disallowed type to a parent node.

Parameters:

  • parent (ast.Node) – The parent node.

  • child (ast.Node) – The child node that is being added to the parent node.
